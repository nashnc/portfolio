## ğŸ“¦ Getting Started

1. **Clone & Install**
   ```bash
   git clone https://github.com/nashnc/portfolio.git
   cd portfolio
   npm install
   ```

2. **Development**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173)

3. **Build & Deploy**
   ```bash
   npm run build
   npm run deploy
   ```
